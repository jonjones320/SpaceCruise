# Space Cruise

## Description: An outerspace flying game in-which the player manuevers a ship to avoid meteors.

## Let's Begin:
1) Make sure you have the [LC-3 Simulator](https://github.com/DimitarKum/LC3-Simulation) (version 3).
2) Save the raw code from this repository and load in LC-3
3) Assemble the code  
4) Move to Simulator window:
-  - Reinitialize machine (second-from-bottom button on the left)
- - Reload object files (circle-arrow button on the left)
- - Press play!  

![simulator window example](/assets/images/how_to_play1.png)  
---
5) Adjust screen zoom, found here:  

![zoom settings menu](/assets/images/zoom_menu.png)  
---
- Zoom as necessary until the bar `|` at the end of `|-SPACE CRUISE-|` is directly under the "C" in "Console (click to focus)".  

![zoom adjustment guide](/assets/images/zoom_alignment_example.png)

6) Now click on the console and press any key to begin!  
---
## How to Play:
- Fly your space ship 'A'
- Move left and right using `a` and `d` keys
- Avoid incoming asteroids '*'   

![play example](/assets/images/spacecruise-ezgif.com-optimize.gif)
---



## Pro Tips:
1) Once the game starts, take a second to figure out the screen; this is an ASCII-animation game--it is jumpy/shaky/looks terrible. Because of this, it will take you a second to track the screen movement, but you will have a few seconds before any asteroids are near.

2) Movement input lags, so just hit `a`/`d` once per frame (multiple inputs will carry into the next frame).
